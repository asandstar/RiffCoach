为什么做这个？

作为一个自学吉他和 ukulele 的人，我发现最大的问题不是没有教程——B站上有无数好视频—— 而是没有结构化的练习追踪。看了很多教程，但不知道自己练了什么、练了多久、哪里卡住了。 RiffCoach 就是为了填补这个空白：把散落的视频教程组织成练习课程，记录每次练习的时间和感受， 用日历看到自己的坚持，用知识库随时查阅乐理。

核心功能

为自学弦乐者设计的完整练习工作流

🎬

B站视频集成

粘贴B站链接自动解析 BV 号，支持选集切换，内嵌播放器边看边练。多代理 CORS 轮换确保稳定获取分集列表。

⏱️

练习计时器

为每节课设定目标时长，开始/暂停/重置，完成后自动记录到练习历史。带节拍器同步，暂停时自动静音。

🎵

内置节拍器

Web Audio API 驱动，BPM 可调，练习时自动同步。完成练习或暂停计时器时自动停止，不会有"幽灵节拍"。

📅

练琴日历

月视图日历展示每日练习时长，类似番茄 Todo。一眼看到自己的坚持轨迹，连续打卡激励持续练习。

📚

内置知识库

24 条乐理知识 + 13 个B站视频教程，覆盖电吉他、木吉他、ukulele。分类筛选，随时查阅，无需联网搜索。

💾

数据备份

一键导出/导入 JSON 数据，深合并迁移机制确保版本升级时不丢失历史记录。完全本地存储，隐私安全。

技术架构

SPA
单页应用架构

客户端路由 navigateTo()，无后端依赖，打开即用。

STORAGE
LocalStorage 持久化

loadData/saveData + mergeDefaults 深合并，支持 schema 迁移。

AUDIO
Web Audio API

OscillatorNode 生成节拍器声音，零音频文件依赖。

CORS
多代理 CORS 轮换

corsproxy.io → allorigins.win → codetabs.com，5s 超时自动切换。
