开始练习 了解功能

为什么做这个？

作为一个自学吉他和 ukulele 的人，我发现最大的问题不是没有教程——B站上有无数好视频—— 而是没有结构化的练习追踪和智能计划。

看了很多教程，但不知道自己练了什么、练了多久、哪里卡住了。 RiffCoach 就是为了填补这个空白：AI 帮你把视频教程和 cover 目标拆成每天能练的任务， 根据你的可用时间、练习历史、BPM 和卡点，生成最小有效练习计划。

核心功能

AI 驱动的完整练习工作流

✨

AI 高效练习计划

根据今日可用时间、精力状态、Cover 目标和最近卡点，生成最小有效练习计划。10-45分钟智能分配。

🎸

Cover 目标管理

把喜欢的歌拆成可追踪的段落进度。记录每个段落的 target BPM、当前干净 BPM、卡点和进度百分比。

📚

资料中心

视频教程、素材篮、已整理练习一站式管理。粘贴链接快速添加，AI 自动整理成练习任务。

🎬

视频学习页

B站视频嵌入 + AI 摘要 + 关键技能标签 + 建议练习方式 + 选集切换（自动获取失败可手动输入）。

📖

目录式知识库

电吉他/木吉他/尤克里里分类知识，包含持琴姿势、基本功、技巧、乐理等，桌面端左侧目录+右侧内容。

🤖

AI 练后反馈

练习完成后，AI 根据 BPM、自评、卡点和练习时长，给出今日总结、可能原因、下次计划和 Cover 进度更新。

📅

周复盘

本周练习统计、Cover 进度变化、技能进步、反复卡点分析、AI 下周建议，一目了然。

⏱️

练习工具

计时器 + 节拍器 + BPM 调节 + 重复次数 + 自评 + 卡点标签 + 备注，完整练习记录。

💾

数据备份

一键导出/导入 JSON 数据，本地存储零依赖，支持旧数据迁移，隐私安全。

使用流程

1
添加资料

粘贴 B站链接或标题，选择用途（关联 Cover / 收藏 / 基础练习），快速加入素材篮。

2
AI 整理

AI 根据标题自动分析，生成 2-4 个建议练习任务，确认后进入已整理练习。

3
生成计划

今日页选择可用时间和精力状态，AI 生成最小有效练习计划。

4
开始练习

进入练习页，计时器 + 节拍器 + 卡点记录，沉浸式练习。

5
AI 反馈

完成练习后，AI 给出今日总结、下次计划和 Cover 进度更新。

6
周复盘

查看本周练习数据，追踪 Cover 进度，调整下周计划。

支持三种乐器

⚡ 电吉他 🎸 木吉他 🌺 尤克里里

每种乐器有专属的视频教程和知识库分类。练习时背景渐变随乐器切换，营造沉浸式练习氛围。

技术架构

Vite
Vite + React + TypeScript

现代化前端工程化，极速开发体验，类型安全。

Zustand
Zustand 状态管理

轻量级状态管理 + LocalStorage 持久化，支持数据迁移。

Tailwind
Tailwind CSS 3

原子化 CSS，自定义主题变量，玻璃拟态设计系统。

Audio
Web Audio API

OscillatorNode 生成节拍器声音，零音频文件依赖。

API
B站 API 集成

BVID 解析、播放器 URL 构建、选集获取（多代理兜底）。

快速开始

# 克隆仓库
git clone https://github.com/asandstar/RiffCoach.git
cd RiffCoach
# 安装依赖
npm install
# 启动开发服务器
npm run dev
# 构建生产版本
npm run build

数据存储在浏览器 LocalStorage，无需后端服务，隐私安全。
