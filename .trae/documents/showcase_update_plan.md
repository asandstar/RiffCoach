# RiffCoach 创意展示页更新计划

## 问题分析

当前 `riff-coach-showcase.html` 存在以下问题，未能准确反映 React 重构版的实际功能：

1. **BPM 旋钮展示不准确**
   - 实际：270° 旋转范围、鼠标拖拽调节、触摸支持、中间输入框可手动输入
   - 当前：静态图示，无交互效果

2. **知识库布局错误**
   - 实际：左侧可折叠导航栏（状态记忆到 localStorage）+ 右侧内容区
   - 当前：简单的左右分栏，无折叠功能

3. **练习页功能不完整**
   - 实际：完整计时器控制（播放/暂停/前进/后退/时间点选择）+ Web Audio API 节拍器
   - 当前：简化的计时器展示

4. **导航入口差异**
   - 实际：知识库入口在"我的"页面（`MePage`），不是底部导航
   - 当前：底部导航显示"资料"包含知识库

## 修改计划

### 文件修改
- `riff-coach-showcase.html` - 完全重写，准确反映 React 项目实际功能

### 修改步骤

1. **BPM 旋钮展示**
   - 实现可交互的旋转旋钮（CSS 旋转 + JavaScript 拖拽）
   - 中间输入框支持手动输入
   - +1/-1 按钮微调
   - SVG 进度环显示当前值比例

2. **知识库布局**
   - 左侧可折叠导航栏（带收起/展开按钮）
   - 双层分类：乐器分类 + 知识分类
   - 右侧知识条目列表和详情展示
   - 导航状态记忆效果

3. **练习页功能**
   - 完整计时器控制（播放/暂停/前进/后退）
   - 时间点选择按钮（5/10/15/20/30/45/60分钟）
   - 节拍器开关
   - 重置功能

4. **导航入口修正**
   - 修正知识库入口描述（从"我的"页面进入）
   - 更新底部导航说明

5. **项目结构更新**
   - 准确反映 `src/` 目录下的文件结构

## 关键功能展示

### BPM 旋钮（参考 BpmKnob.tsx）
- 旋转范围：-135° 到 135°（共 270°）
- 值范围：40-200 BPM
- 交互方式：鼠标拖拽、触摸滑动、输入框输入
- 视觉：玻璃拟态旋钮 + SVG 进度环 + 指针

### 知识库（参考 KnowledgePage.tsx）
- 乐器分类：全部、电吉他、木吉他、尤克里里
- 知识分类：入门、基本功、技巧、乐理
- 导航栏：可折叠（宽度从 256px 收缩到 56px）
- 状态记忆：localStorage 保存折叠状态

### 练习页（参考 PracticePage.tsx）
- 计时器：播放/暂停、前进/后退 60 秒、时间点选择
- 节拍器：Web Audio API 实现，无音频文件依赖
- BPM 旋钮：集成式调节
- 数据记录：重复次数、自评、卡点标签、备注
