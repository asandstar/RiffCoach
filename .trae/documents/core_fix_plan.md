# 核心问题修复计划

## 一、问题分析

### 问题1：视频选集标题显示不全
- **现象**：选集只显示"第1集"、"第2集"等通用名称，没有显示实际分集标题
- **根因**：
  1. B站API代理请求失败时回退到fallback数据（只有通用名称）
  2. defaultData中的视频资源可能没有完整的分集标题

### 问题2：选集没有全部显示
- **现象**：只显示P1-P5，但提示"共10集"
- **根因**：`VideoEpisodePicker` 中 `max-h-48`（192px）限制了高度，超过5个分集就会被截断

### 问题3：按钮功能逻辑缺失
- **现象**：用户反馈开始按钮等功能没加上
- **根因**：需要检查各页面的按钮onClick事件是否完整

### 问题4：桌面端布局问题（最严重）
- **现象**：电脑上只能看到中间很小一部分内容
- **根因**：`PageShell` 使用 `max-w-lg`（512px）限制内容宽度，没有利用桌面端屏幕空间

## 二、修复方案

**修改文件**：

**具体修改**：
