# 重写 riff-coach-showcase.html 更新计划

## 问题分析

当前 `riff-coach-showcase.html` 的内容与实际 React 重构版项目不符，主要问题：
1. BPM 调节展示缺少旋转旋钮效果
2. 知识库展示过于简单，缺少左侧可折叠导航栏
3. 练习页功能不完整（缺少时间点选择、卡点详情等）
4. 整体布局与实际 React 应用存在偏差

## 重构后的项目核心特性（需准确展示）

### 1. BpmKnob 组件
- 270° 旋转范围（-135° 到 +135°）
- 鼠标拖拽 + 触摸支持
- 输入框手动输入 + 验证（40-200）
- 步进按钮（+/-1）
- SVG 进度环可视化

### 2. KnowledgePage 知识库
- 桌面端左侧可折叠导航栏（状态记忆到 localStorage）
- 乐器分类：全部 / 电吉他 / 木吉他 / 尤克里里
- 知识分类：入门 / 基本功 / 技巧 / 乐理
- 条目详情：一句话解释、关联技能、常见错误、练习建议、详细内容

### 3. PracticePage 练习页
- 计时器：播放/暂停/前进(+60s)/后退(-60s)/时间点选择(5/10/15/20/30/45/60分)
- BPM 旋钮旋转调节
- 节拍器（Web Audio API OscillatorNode）
- 重复次数、自评(1-5星)、卡点标签、备注
- 卡点详情选择（如"节奏不稳"下有具体问题选项）

### 4. ResourcePage 资料中心
- 4个标签页：最近使用 / 素材篮 / 视频教程 / 已整理练习
- 全局搜索功能
- AI 整理按钮、转为练习任务按钮

### 5. CoverPage Cover 目标管理
- 添加/删除 Cover 目标
- 段落进度追踪（Intro/Verse/Full）
- AI 拆解练习路径

### 6. MePage 我的页
- 快速入口：知识库、练琴日历
- 数据管理：导出/导入 JSON
- 60秒演示路径

### 7. VideoStudyPage 视频学习页
- B站视频嵌入 + 选集切换
- AI 摘要、关键技能、建议练习方式
- 常见错误提示
