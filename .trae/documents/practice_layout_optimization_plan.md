# 练习工具布局优化计划

## 问题分析

用户反馈练习工具界面存在以下问题：

1. **BPM数字和调整按钮重复显示**：BpmKnob组件内部已经包含了BPM数字和+/-按钮，但外层PracticePage和VideoStudyPage又额外显示了一套，造成重复
2. **布局不合理**：播放按钮和时长选择在右侧，计时器在左侧，操作流程不连贯
3. **功能分区不清晰**：节拍器开关和音色选择位置分散

## 优化方案

### 布局调整

**左侧区域（计时器区域）**：
- 计时器圆环（显示当前时间）
- 播放/暂停按钮（大按钮）
- 前进/后退按钮
- 正计时/倒计时切换按钮
- 时长选择按钮（仅倒计时模式显示，包含常用时间和自定义输入）

**右侧区域（节拍器区域）**：
- 节拍器开关按钮（♪图标）
- 拍数可视化显示（1-4/6个圆点）
- 拍号选择（2/4、3/4、4/4、6/8）
- BpmKnob组件（自带BPM数字和+/-按钮）
- 音色选择（电子、木鱼、点击、蜂鸣）

### 需要修改的文件

| 文件 | 修改内容 |
|------|----------|
| `src/pages/PracticePage.tsx` | 重构练习工具卡片布局，删除重复的BPM数字和按钮 |
| `src/pages/VideoStudyPage.tsx` | 同步修改练习工具面板布局 |
| `src/components/BpmKnob.tsx` | 保持不变（已有完整功能） |
| `src/hooks/usePractice.ts` | 添加自定义时长输入支持 |

### 修改步骤

1. **修改 PracticePage.tsx**：
   - 将播放/暂停、前进/后退按钮移到左侧计时器下方
   - 将时长选择按钮移到左侧，仅在倒计时模式显示
   - 删除右侧重复的BPM数字显示和+/-按钮
   - 将节拍器开关、拍数显示、拍号选择、音色选择整合到右侧
   - 添加自定义时长输入框（倒计时模式）

2. **修改 VideoStudyPage.tsx**：
   - 同步PracticePage的布局调整
   - 删除重复的BPM数字和按钮

3. **修改 usePractice.ts**：
   - 添加自定义时长设置方法

### 风险处理

- 修改布局可能影响响应式适配，需测试移动端显示效果
- 删除重复代码需确保功能完整性，测试所有按钮交互
- 自定义时长输入需添加输入验证，确保输入有效数值

## 预期效果

- 左侧：计时器 + 播放控制 + 时长选择（倒计时模式）
- 右侧：节拍器开关 + 拍数显示 + 拍号选择 + BpmKnob + 音色选择
- 无重复元素，布局清晰，操作流程顺畅
