# RiffCoach · AI 高效练习教练

> 把喜欢的歌，拆成每天能练的任务。每天只有 20 分钟，也能离你的 cover 目标更近一步。

## 🏆 项目简介

RiffCoach 是一款面向业余弦乐学习者的 **AI 高效练习教练**，支持电吉他、木吉他、尤克里里三种乐器。通过 AI 驱动的练习计划生成、Cover 目标管理、资料中心、视频学习、知识库和周复盘，帮助用户系统化地追踪练习进度，克服自学瓶颈。

**核心价值：**

| 维度 | 说明 |
|------|------|
| 🎯 **目标导向** | 从喜欢的歌出发，拆成可追踪的段落进度 |
| ⏱️ **时间高效** | 每天 20 分钟也能产生有效练习成果 |
| 🤖 **智能计划** | AI 根据时间、状态、卡点生成最小有效练习计划 |
| 📊 **数据驱动** | 记录 BPM、自评、卡点，量化进步过程 |

## ✨ 核心功能

### 🤖 AI 高效练习计划
根据今日可用时间、精力状态、Cover 目标和最近卡点，智能生成最小有效练习计划。支持少于 15 分钟专项练习、20-30 分钟完整练习、超过 40 分钟深度训练等多种模式。

### 🎸 Cover 目标管理
把喜欢的歌拆成可追踪的段落进度，记录每个段落的练习数据（进度百分比、Target BPM、Clean BPM、卡点标签）。AI 自动分析并生成练习路径。

### 📚 资料中心
视频教程、素材篮、已整理练习一站式管理。四大标签页：最近使用、素材篮、视频教程（B站视频聚合）、已整理练习。支持粘贴 B站/YouTube 链接自动解析。

### 🎬 视频学习页
深度集成 B站视频，提供沉浸式学习体验。核心功能：B站播放器嵌入、AI 摘要、关键技能标签、选集切换（支持自动获取和手动输入）、建议练习方式、**练习模式（保留视频+完整练习工具）**。桌面端两栏布局，视频与练习工具并排显示，充分利用屏幕空间。

### 📖 目录式知识库
专业的乐器知识体系，支持多乐器分类浏览（电吉他、木吉他、尤克里里）。核心功能：
- **搜索功能**: 支持按标题、摘要、标签、关联技能实时搜索
- **收藏系统**: 一键收藏知识点，支持收藏筛选
- **阅读记录**: 自动记录已读状态和阅读时间
- **资料来源**: 每篇文章标注参考资料（书籍、视频、教程）
- **相关推荐**: 智能推荐关联知识点和视频教程
- **左侧可折叠导航栏**: 导航状态记忆到 localStorage
- **65+ 篇专业文章**: 涵盖入门、基本功、技巧、乐理、练习心态等多个维度

### ⏱️ 练习工具
完整的练习辅助工具面板，整合为一个卡片：
- **计时器**: 支持正计时/倒计时双模式、播放/暂停、前进/后退、时间点快速选择（5/10/15/20/30/45/60分钟）、环形进度条
- **节拍器**: Web Audio API 实时音效，支持2/4、3/4、4/4、6/8拍号切换，视觉拍数显示，重音高亮，四种音色选择（电子、木鱼、点击、蜂鸣）
- **BPM 旋钮**: 270° 旋转交互，支持拖拽调节，键盘快捷键（+/-）微调
- **键盘快捷键**: 空格（播放/暂停）、B（节拍器开关）、R（重置）、方向键（前进/后退）、+/-（BPM 增减）
- **练习模板**: 保存/加载/删除常用练习配置，支持本地存储持久化，内置4个预设模板（热身练习、节奏训练、慢速练习、3/4拍练习）
- **自评系统**: 1-5星评分，卡点标签，练习备注
- **完成动画**: 练习完成后庆祝动画效果

### 🤖 AI 练后反馈
练习完成后 AI 智能分析，核心功能：
- **今日总结**: 自动生成练习成果总结
- **BPM 分析**: 尝试 BPM、最高干净 BPM 统计
- **下次练习计划**: 基于当前状态生成针对性建议
- **Cover 进度更新**: 自动更新目标完成度
- **AI 智能推荐**: 根据练习数据推荐视频、练习任务和知识点

### 🏆 成就系统
12种成就激励用户持续练习：
- **初出茅庐**: 完成第一次练习
- **坚持不懈**: 连续练习7天
- **时间大师**: 累计练习10小时
- **速度之星**: BPM达到120
- **完美主义者**: 自评5星次数达到10次
- 以及更多成就等待解锁

### 📅 周复盘
每周练习数据汇总分析：总练习分钟、练习天数、平均 BPM、Cover 进度变化、反复卡点分析。

### 部署说明
项目支持两种部署方式：

**GitHub Pages**：构建产物输出到 `docs/` 目录。在 GitHub 仓库设置中配置 Pages 从 `main` 分支的 `/docs` 目录部署。

**Cloudflare Pages**：支持 Pages Functions（后端 API）、Workers AI、D1 数据库等服务端能力。详见 [CLOUDFLARE_DEPLOYMENT.md](./CLOUDFLARE_DEPLOYMENT.md)。

项目相关 Cloudflare 配置：
- `functions/api/feedback.ts` — AI 反馈 API（当前 Mock，未来接入 Workers AI）
- `wrangler.toml` — Pages 项目配置
- `public/_redirects` — SPA 路由重定向
- `npm run build:cloudflare` — Cloudflare 构建命令（base=/，输出到 dist/）

## 🛠️ 技术架构
