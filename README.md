## 🏆 项目简介

RiffCoach 是一款面向业余吉他学习者的 **AI 高效练习教练**。通过 AI 驱动的练习计划生成、Cover 目标管理、资料中心、视频学习、知识库和周复盘，帮助用户系统化地追踪练习进度，克服自学瓶颈。

### 核心价值

| 维度 | 说明 |
|------|------|
| 🎯 **目标导向** | 从喜欢的歌出发，拆成可追踪的段落进度 |
| ⏱️ **时间高效** | 每天 20 分钟也能产生有效练习成果 |
| 🤖 **智能计划** | AI 根据时间、状态、卡点生成最小有效练习计划 |
| 📊 **数据驱动** | 记录 BPM、自评、卡点，量化进步过程 |

## ✨ 核心功能详解

### 🤖 AI 高效练习计划

根据今日可用时间、精力状态、Cover 目标和最近卡点，智能生成最小有效练习计划。

**智能规则引擎：**
- 少于 15 分钟：只安排一个最重要卡点的专项练习
- 20-30 分钟：热身 → 卡点专项 → 串联练习 → 简单复盘
- 超过 40 分钟：完整训练 + 录音反馈 + 深度复盘
- 状态很累/手指酸：不建议冲 BPM，安排轻松复习内容
- 有 Cover 项目：优先选择最高进度未完成段落

**输出内容：**
- 今日最小有效目标
- 为什么今天练这个（基于练习历史分析）
- 分步骤计划（时长 + BPM + 练习说明）
- 今天不建议做什么（避免无效练习）
- 完成标准（明确的验收条件）

---

### 🎸 Cover 目标管理

把喜欢的歌拆成可追踪的段落进度，记录每个段落的练习数据。

**段落追踪维度：**
- 📈 进度百分比（0-100%）
- 🎵 Target BPM（目标速度）
- ✨ Clean BPM（最高干净演奏速度）
- 🎯 卡点标签（标记难点）

**AI 拆解功能：**
- 输入歌曲名称或视频链接
- AI 自动分析并生成练习路径
- 建议练习顺序和重点

---

### 📚 资料中心

视频教程、素材篮、已整理练习一站式管理。

**四大标签页：**
1. **🕐 最近使用** — 快速回到上次练习的内容
2. **📁 素材篮** — 收藏的视频、乐谱、练习材料
3. **▶ 视频教程** — B站视频聚合，支持电吉他/木吉他/尤克里里分类
4. **📖 已整理练习** — AI 自动整理后的结构化练习任务

**快速添加：**
- 粘贴 B站 / YouTube 链接自动解析
- 搜索关键词快速查找（如：爬格子、交替拨弦、五声音阶）
- 选择用途：关联 Cover / 收藏 / 基础练习

---

### 🎬 视频学习页

深度集成 B站视频，提供沉浸式学习体验。

**核心功能：**
- 📺 B站播放器嵌入（自适应宽屏）
- 🤖 AI 摘要（自动分析视频内容）
- 📝 关键技能标签（提取视频核心知识点）
- 📑 选集切换（自动获取，失败可手动输入）
- 💡 建议练习方式（基于视频内容的练习建议）

**多代理选集获取：**
- 自动获取 B站视频选集列表
- 支持多种 API 代理兜底
- 失败时提供手动输入备选

---

### 📖 目录式知识库

专业的乐器知识体系，支持多乐器分类浏览。

**双层分类结构：**
- **乐器分类**：电吉他、木吉他、尤克里里、全部
- **知识分类**：入门、基本功、技巧、乐理

**知识条目包含：**
- 一句话解释
- 难度星级（1-5星）
- 关联技能标签
- 常见错误提示
- 练习建议
- 详细内容（段落、列表、提示）

**交互特性：**
- 左侧可折叠导航栏（桌面端）
- 导航状态记忆到 localStorage
- 移动端自适应布局

---

### ⏱️ 练习工具

完整的练习辅助工具面板。

**核心工具：**
- ⏰ **计时器** — 播放/暂停、前进/后退 60秒、时间点选择（5/10/15/20/30/45分钟）
- 🎵 **节拍器** — Web Audio API 生成，无音频文件依赖，支持开关控制
- 🔘 **BPM 旋钮** — 270° 旋转范围（40-200 BPM），支持拖拽、手动输入、±1 微调
- ⭐ **自评** — 1-5星评价系统
- 🏷️ **卡点标签** — 预设常见卡点（节奏不稳、换和弦慢、手指僵硬等）+ 详细问题选择
- 📝 **备注** — 自由文本记录练习感受

---

### 🤖 AI 练后反馈

练习完成后 AI 智能分析，给出个性化反馈。

**反馈维度：**
- 📋 今日总结（基于练习数据的分析）
- 🎵 尝试 BPM（练习过程中的尝试速度）
- ✨ 最高干净 BPM（能够稳定演奏的最高速度）
- 📅 下次练习计划（基于本次表现的建议）
- 📈 Cover 进度更新（自动更新相关段落进度）

---

### 📅 周复盘

每周练习数据汇总分析。

**数据统计：**
- ⏱️ 总练习分钟
- 📆 练习天数
- 🎵 平均 BPM
- 📈 Cover 进度变化
- 🏷️ 反复卡点分析

## 🚀 快速开始

### 在线体验

- **主应用**: https://asandstar.github.io/RiffCoach/
- **创意展示**: https://asandstar.github.io/RiffCoach/riff-coach-showcase.html

### 本地开发

```bash
# 克隆仓库
git clone https://github.com/asandstar/RiffCoach.git
cd RiffCoach

# 安装依赖（建议使用 npm >= 10）
npm install

# 启动开发服务器（端口 5173）
npm run dev

# 构建生产版本
npm run build

# 预览生产版本
npm run preview
```

### 首次使用

1. 打开应用后点击「加载演示数据」体验完整功能
2. 在「今日」页面选择可用时间和精力状态
3. 点击「生成今日计划」获取 AI 推荐的练习安排
4. 点击「一键开始练习」进入练习页
5. 完成练习后查看 AI 反馈和周复盘

## 🎯 使用流程

```
1. 添加资料          → 粘贴B站链接/搜索教程 → 加入素材篮
       ↓
2. AI 整理           → AI分析视频内容 → 生成2-4个练习任务 → 进入已整理练习
       ↓
3. 生成计划          → 选择时间/状态 → AI生成最小有效计划 → 查看今日目标
       ↓
4. 开始练习          → 计时器+节拍器+BPM旋钮 → 记录自评和卡点 → 完成练习
       ↓
5. AI 反馈           → 查看今日总结 → 了解尝试BPM和干净BPM → 获取下次计划
       ↓
6. 周复盘            → 查看本周数据 → 分析反复卡点 → 调整下周计划
```
